# Ultimate AI Fake Review Detection System

This project is a multilingual AI-powered fake review detection system using Transformer-based BERT NLP.

## Features

- Fake Review Detection
- Sentiment Analysis
- Emotion Detection
- Explainable AI
- Language Detection
- Auto Summarization
- Keyword Highlighting
- Dashboard Analytics
- Multi-Modal AI

## Technologies Used

- Python
- Streamlit
- Plotly
- Pandas
